# FleetFlow

A distributed logistics and delivery orchestration platform built using:

- ASP.NET Core
- Angular
- PostgreSQL
- RabbitMQ
- Redis
- SignalR
- Docker

## Goals

- Event-driven architecture
- Real-time order tracking
- Intelligent rider assignment
- Distributed services
- Production-ready deployment

## Services

- API Gateway
- Order Service
- Delivery Service
- Notification Service
